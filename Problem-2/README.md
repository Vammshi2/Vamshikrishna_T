
# Odd Number Series Generator

## Problem Statement

Given a number `a`, generate the first `a` odd numbers starting from 1.

### Example

| Input | Output         |
|-------|----------------|
| 1     | 1              |
| 2     | 1, 3           |
| 3     | 1, 3, 5        |
| 4     | 1, 3, 5, 7     |
| x     | 1, 3, 5, 7,... |
